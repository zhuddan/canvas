# 渲染元素

所有渲染元素都都是继承于 `Renderable`抽象类，`Renderable`抽象类包含了所有渲染元素的公共属性和方法。


## 属性
### position
| 属性    | 类型 |
| --- | ------- |
| x  | number    |
| y  | number    |


#### 初始位置

`position` 和 `x/y` 均可以控制渲染元素的坐标。你可以在初始化渲染元素时候传入`postion`或者`x/y`指定元素的初始位置，若你未指定，则默认为(0,0)，元素会在app的右上角显示。

:::warning

初始化渲染元素传入的 <strong>`position`</strong> 的权重高于<strong> `x/y` </strong>，这意味着同时传入`position`和 `x/y`的情况下，只有`position`生效（虽然我相信没有人会这么做）。

:::

- 修改位置

你可以使用`renderable.position.set(x,y)`设置元素位置，如果`x`和`y`相同，则第二个参数可以省略。
或者`renderable.position={x,y}`，如果你只想改变横坐标坐标的话，可以直接使用`renderable.position.x = x`或者`renderable.position.x = x`, 纵坐标同理。

<Playground>
```ts
import { App, Shape } from '@zd~/canvas'

const app = new App({
  backgroundColor: '#60a5fab0',
  resizeTo: window,
})

document.body.appendChild(app.canvas)

// 我的初始位置是 0，0
const shape = new Shape()
shape.beginPath().rect(0, 0, 60, 60).fill('#FFCC80')

// 我的初始位置是 80, 80
const shape2 = new Shape({
  position: {
    x: 80,
    y: 80,
  }
})
shape2.beginPath().rect(0, 0, 60, 60).fill('#4DB6AC')

// 我的初始位置是 180, 10
const shape3 = new Shape({
  position: {
    x: 180,
    y: 10,
  }
})
shape3.beginPath().rect(0, 0, 60, 60).fill('#CDDC39')

app.add(shape, shape2, shape3)

app.ticker.add(()=>{
  // 动态修改
  shape3.y++
  if(shape3.y>=app.height){
    shape3.y = 10
  }
})

```
</Playground>
