渲染元素

所有渲染元素都都是继承于 Renderable抽象类，Renderable抽象类包含了所有渲染元素的公共属性和方法。

属性​

position​

属性类型
xnumber
ynumber

初始位置​

position 和 x/y 均可以控制渲染元素的坐标。你可以在初始化渲染元素时候传入postion或者x/y指定元素的初始位置，若你未指定，则默认为(0,0)，元素会在app的右上角显示。

warning

初始化渲染元素传入的 position 的权重高于 x/y ，这意味着同时传入position和 x/y的情况下，只有position生效（虽然我相信没有人会这么做）。

  • 修改位置

你可以使用renderable.position.set(x,y)设置元素位置，如果x和y相同，则第二个参数可以省略。 或者renderable.position={x,y}，如果你只想改变横坐标坐标的话，可以直接使用renderable.position.x = x或者renderable.position.x = x, 纵坐标同理。
