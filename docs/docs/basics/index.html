渲染元素

所有渲染元素都都是继承于 Renderable 抽象类，他实现了最基础的形变与变换操作。

position 坐标​

控制一个渲染元素的坐标位置

初始坐标​

使用position属性控制元素的初始位置。

const shape = new Shape({
  position: {
    x: 100,
    y: 100,
  }
})

也可以使用x或者y属性控制元素的初始位置。

const shape = new Shape({
  x: 100,
  y: 100,
})

⚠️ 同时使用position和x或者y属性的情况下只有position生效。

const shape = new Shape({
  position: {
    x: 100,
    y: 100,
  },
  x: 50, // ==> dont work
  y: 50, // ==> dont work
})

修改坐标​

直接给position赋值。

shape.position = {
  x: 0,
  y: 0
}

或者给position.x或position.y赋值或者给x进行赋值y都可以修改坐标。

shape.position.x = 0
shape.position.y = 0
// or
shape.x = 0
shape.y = 0

另外你还可以使用 position.set() 修改坐标。如果x和y相同，则可以省略第二个参数。

shape.position.set(0, 0)
// or
shape.position.set(0)

visible 可见性​

控制一个渲染元素的是否显示，接收一个boolean 值。默认`true。

const shape = new Shape({
  visible: false
})
shape.visible = true

shadowType 阴影​

元素阴影。是shadowColor shadowOffsetX shadowOffsetY 和 shadowBlur组合起来的一个对象。

⚠️你至少应该提供一个color属性。此外阴影可能会造成性能问题，我们会在后文中解释。

const shape = new Shape({
  shadow: {
    color: 'rgba(0, 0, 0, 1)',
    x: 5,
    y: 5,
    blur: 0,
  },
})

shape.shadow.color = 'red'
shape.shadow.blur = 10

anchor 锚点​

控制元素的中心点 (0~1)，默认(0,0)表示右上角。

如果一个元素的中点设置为(0.5,0.5)则表示中心点在元素中间。

如果一个元素的中点设置为(1,1)则表示中心点在元素右下角。

这个属性可以更方便的调整元素的位置，或者旋转缩放中心。

例如我期望一个元素位于app的中心位置, 我只用设置他的坐标为(app.width/2, app.height/2)，并且把anchor设置为0.5。
