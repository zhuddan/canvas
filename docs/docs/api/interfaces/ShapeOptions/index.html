anchor?​

optional anchor: number | PointData

单位矩阵变化中心，默认是[0,0]表示左上角，[1,1]表示右下角，[0.5,0.5]表示中心

此属性可以很好的控制元素的锚点，不用计算相对位置

例如：一个500x500的画布上需要绘制一个上下居中的图片

const text = new Picture('demo.png',{
 anchor: 0.5
})
