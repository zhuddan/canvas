安装​

web 或者 uni-app​

  • npm
  • Yarn
  • pnpm
npm install @zd~/canvas
yarn add @zd~/canvas
pnpm add @zd~/canvas
tip

微信小程序使用 npm包 是个十分麻烦的过程, 建议您使用原生web或者uni-app构建应用。 如果您确实需要使用原生微信小程序构建npm, 请参考微信小程序npm构建 。不过更推荐这么做：

微信小程序​
